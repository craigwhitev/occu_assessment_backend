## Description

This project is a backend application built with [NestJS](https://nestjs.com/), a progressive Node.js framework for building efficient and scalable server-side applications. The application provides APIs for managing products, including searching, creating, updating, and deleting products.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Environment Variables

Create a .env file in the root of your project and add the following environment variables:

```
PORT=3000
ALLOWED_ORIGINS=https://example.com,https://test.com
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=
DB_DATABASE=occu_test
```

## Swagger

The API documentation is available at /api endpoint. It is generated using Swagger.
